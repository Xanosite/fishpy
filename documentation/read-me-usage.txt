Application data file:
  location: application folder /data/application-data.dat
  IAID = A numerical identifier for your application, unique to the server.
  IADescription = Description of your application, will be viewable from client view.
  IAName = A name for your application, unique to the server. Viewable from client view.
  Host = server
  Port = api port, default 28192

  format as follows
{
  "IAID":"1337",
  "IADescription":"An odd choice of filling for a pie.",
  "IAName":"FishPy",
  "Host":"192.168.1.1",
  "Port":"28192"
}
